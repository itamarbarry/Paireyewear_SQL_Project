# Pair Eyewear SQL Project

### Project Description  
This project was completed as part of the **"Data Bases"** B.Sc. course in the **Industrial Engineering & Management Department** at **Ben-Gurion University**.  

It focuses on business analysis of data from [**Pair Eyewear**](https://paireyewear.com), a website that sells customizable glasses and sunglasses. The main process represented is the purchase of various types of eyewear and accessories.  

We designed a customized database, performed advanced SQL queries to extract insights, and created analytical dashboards to support data-driven business decisions.
